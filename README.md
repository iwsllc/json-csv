## Dev Setup

```bash
# install everything for all workspaces (Recommended: Node 20)
npm i

# build the packages
npm run build

# run the samples
npm start -w samples

# run the tests with current runtime
npm test

# for older versions of node <= 12, cd to the test-runner directory
cd test-runner && npm test
```
